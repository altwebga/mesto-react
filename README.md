# Проект: Место
Сервис Mesto: интерактивная страница, куда можно добавлять фотографии, удалять их и ставить лайки.


### Технологии
* React
* HTML5
* CSS3
* JS
* Медиазпросы
* Структура проекта организована по методологии БЭМ
* Верстка выполнена с использованием grid и flexbox.
